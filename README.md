  and admission trends using pivot Tables, pivot charts, and Slicers

-Microsoft Excel
-Pivot Table
-Pivot Charts
-Slicers

## Dashboard Features
-Daily Pateint Count
-Average Wait Time
-Patient Satisfaction Score
-Pateint Admission Analysis
-Interactive Filters using Slicers
